RSA Online New South Wales - Reference Resources

The following resources relate to RSA in New South Wales. They’re provided here in case you want to find out more about Online RSA training, or RSA-related activities and regulations.

Fact Sheet - Special licence conditions for premises in Kings Cross (NSW)

File size: 1.74 MB (Modified: 25/06/2016 11:36 pm) Download file

This fact sheet details special conditions surrounding premises in Kings Cross, NSW. (Office of Liquor, Gaming & Racing, December 2014).

NSW Liquor Act 2007

File size: 3.39 MB (Modified: 25/06/2016 11:40 pm) Download file

The actual Liquor Act 2007 (No. 90), revised 19 Feb 2016. Includes a lot of details on licensing, responsible service, and details of regulations.

Liquor Promotion Guidelines (NSW)

File size: 1.52 MB (Modified: 25/06/2016 11:46 pm) Download file

This provides important information on promoting liquor, and the responsibility of staff and licensees to have responsible attitudes and practices regarding the promotion and sale of alcohol. It is a comprehensive and useful document, and outlines various principles relating to minors, discounts, rate of alcohol consumption, harm minimisation, the public interest, and more. (NSW Office of Liquor, Gaming & Racing, July 2013).

Liquor Signage (NSW)

File size: 3.78 MB (Modified: 25/06/2016 11:49 pm) Download file

This contains information on liquor signage for various premises, including clubs, hotels, small bars, packaged liquor outlets / bottle shops and more. (NSW Liquor and Gaming, May 2016).

NSW Liquor Regulation 2008

File size: 2.71 MB (Modified: 25/06/2016 11:54 pm) Download file

This document outlines regulations that relate to the application of the NSW Liquor Act (2007), covering topics such as the sale, supply, service and promotion of liquor. (NSW Government, version 1.3.2016).

Fact Sheet - Underage drinking laws (NSW)

File size: 225.64 kB (Modified: 25/06/2016 11:59 pm) Download file

This fact sheet  provides key information on responsible adults, proof of age documents, and obligations of licensees and staff with regard to underage drinking. It contains a information on variety of situations and what is required (e.g. minors being in, or passing through, the bar area of a hotel or licensed venue). NSW Office of Liquor, Gaming and Racing, Sept 2012).

Licence conditions regarding Outlaw Motorcycle Gangs (OMCG)

File size: 452.38 kB (Modified: 26/06/2016 12:02 am) Download file

This contains various information about refusal of entry and service for persons wearing prohibited items (e.g. insignia or emblems of motorcycle clubs or gangs) in Australian states. (eLearning Portal 2014, based upon government resources).

National Drug Strategy - Household Survey (2013)

File size: 2.23 MB (Modified: 26/06/2016 12:07 am) Download file

This report contains information on various types of drug use, including alcohol. It contains useful information on alcohol use in the general population, usage trends, risks and harm, and more. It also includes state and territory comparisons. (Australian Institute of Health and Welfare, 2013)

Preventing Intoxication

File size: 95.52 kB (Modified: 26/06/2016 12:09 am) Download file

This information is outlines measures for preventing intoxication, primarily focused on liquor promotions. (Liquor & Gaming NSW, 2016).

Prevention of Intoxication on Licensed Premises (NSW)

File size: 314.07 kB (Modified: 26/06/2016 12:12 am) Download file

This is a concise set of guidelines, covering topics such as the law, monitoring patron consumption and behaviour, and planning to prevent intoxication (NSW Justice, March 2015).
